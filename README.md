# 인사자문위원회 비밀투표 앱

모바일에서 바로 열리도록 만든 웹앱입니다. 앱스크립트 화면 대신 일반 웹서버로 동작하며, 선생님은 휴대폰 브라우저에서 링크에 접속해 직위/성함/핸드폰번호 뒷자리 4개 확인 후 우선순위 교과와 서명을 제출합니다.

## 실행

```bash
cd /Users/oks/Documents/Codex/2026-06-22/new-chat-2/outputs/hr-secret-vote
npm start
```

`npm`이 없는 환경에서는 아래처럼 실행해도 됩니다.

```bash
node server.js
```

브라우저에서 `http://localhost:4173`을 엽니다.

관리자 화면 비밀번호는 설정된 운영 비밀번호를 사용합니다. 실제 운영 시에는 아래처럼 실행할 때 원하는 비밀번호로 지정할 수 있습니다.

```bash
ADMIN_PASSWORD="원하는관리자비밀번호" npm start
```

## 운영 흐름

1. 관리 탭에서 로그인합니다.
2. 투표 안건, 입력 순위 수, 투표 열림/마감 상태를 설정합니다.
3. 직위/성함/핸드폰번호 뒷자리 4개를 미리 등록합니다.
4. 선생님들에게 접속 링크를 안내합니다.
5. 투표 종료 후 관리 탭에서 참여자 CSV와 투표 CSV를 내려받습니다.

## 비밀투표 처리 방식

- 참여자 명단에는 제출 여부와 서명만 저장합니다.
- 투표 내용에는 접수번호, 제출시각, 우선순위만 저장합니다.
- 서버 데이터 안에서 특정 선생님의 이름과 투표 내용이 직접 연결되지 않도록 분리했습니다.

## 휴대폰에서 사용하려면

회의실 내부 테스트는 같은 와이파이에 연결된 컴퓨터의 내부 IP와 포트로 접속할 수 있습니다. 실제 회의 운영은 HTTPS가 제공되는 Render, Railway, Fly.io 같은 Node 호스팅에 올리는 방식을 권장합니다.

배포 시 데이터 파일은 `data/ballot-data.json`에 저장됩니다. 서버를 재배포할 때 이 파일이 지워지는 호스팅이라면 영구 디스크 옵션을 켜야 합니다.

다른 장소와 외부망에서도 열리게 하려면 [DEPLOY.md](/Users/oks/Documents/Codex/2026-06-22/new-chat-2/outputs/hr-secret-vote/DEPLOY.md)를 참고해 인터넷 서버에 배포하세요.
